# Display gas price in sidebar

## Description

Add a gas price indicator to the sidebar so users can see current network gas costs at a glance. The gas fee store already exists and provides slow/average/fast estimates - we just need to display this information in the UI.

Users need visibility into gas prices to:
- Decide when to submit transactions
- Understand if the network is congested
- Plan timing for expensive operations

## Acceptance Criteria

- [ ] Gas price is displayed in the sidebar area
- [ ] Shows at least one tier (average or fast) in gwei
- [ ] Updates automatically when gas fee store updates
- [ ] Shows loading state while fetching
- [ ] Handles error state gracefully if fetch fails

## Notes

**Existing Infrastructure:**
- [[web/src/lib/core/connection/gasFee.ts]] - GasFeeStore with slow/average/fast estimates
- Store provides `GasPriceEstimates` with `maxFeePerGas` values

**Design Considerations:**
- Keep it compact for sidebar real estate
- Consider showing all 3 tiers or just average with hover for details
- Format in gwei for readability
